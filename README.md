# 📝 周报 | Weekly Report

一个简约优雅的个人周报网站，采用 Apple 设计风格。

![Apple Design](https://img.shields.io/badge/Design-Apple%20Style-black?style=for-the-badge)
![Pure Frontend](https://img.shields.io/badge/Stack-HTML%2FCSS%2FJS-orange?style=for-the-badge)
![No Backend](https://img.shields.io/badge/Backend-Not%20Required-green?style=for-the-badge)

## ✨ 功能特性

### 📊 核心功能
- **周视图导航** - 按周浏览和管理周报，支持快速切换
- **完成事项** - 详细记录本周完成的任务
  - 标题与详细内容
  - 分类标签（工作/学习/生活/项目/其他）
  - 优先级设置
  - 自定义颜色标记
  - 完成进度追踪
  - 开始/完成日期
  - 自定义标签
- **收获心得** - 记录学习收获和心得体会
- **个人笔记** - 上传和管理笔记文件
  - 支持多种格式（txt/md/pdf/doc/图片等）
  - 本地存储，安全可靠
- **历史周报** - 时间线形式展示历史记录
- **数据统计** - 可视化展示累计数据趋势

### 🎨 设计特色
- **Apple 设计语言** - 简约、优雅、克制
- **精致配色** - 苹果系统色彩，克制使用
- **流畅动画** - 自然舒适的交互体验
- **响应式布局** - 完美适配各种设备

### 💾 数据管理
- **本地存储** - 所有数据保存在浏览器 localStorage
- **无需登录** - 纯前端应用，开箱即用
- **隐私安全** - 数据仅存储在本地

## 🚀 快速开始

### 在线访问
访问 [GitHub Pages 地址] 即可使用

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/你的用户名/weekly-report.git

# 进入目录
cd weekly-report

# 使用任意 HTTP 服务器运行
python3 -m http.server 8080
# 或
npx http-server -p 8080

# 浏览器访问
# http://localhost:8080
```

### 直接打开
直接双击 `index.html` 文件即可使用（推荐使用现代浏览器）

## 📖 使用指南

### 添加事项/收获
1. 点击顶部「添加事项」或「记录收获」按钮
2. 填写标题（必填）和详细内容
3. 选择分类和优先级
4. 添加自定义标签（可选）
5. 设置完成进度和日期
6. 选择颜色标记
7. 点击保存

### 上传笔记
1. 点击「上传」按钮
2. 选择文件（支持多选）
3. 文件将安全存储在本地
4. 点击笔记卡片可下载

### 浏览历史
- 点击侧边栏「历史」查看所有周报
- 点击任意周报可快速跳转
- 使用顶部的左右箭头切换周次

### 查看统计
- 点击侧边栏「统计」
- 查看累计数据和趋势图

## 🎯 功能详解

### 事项属性

| 属性 | 说明 |
|------|------|
| 标题 | 简短描述（必填） |
| 详细内容 | 补充说明、心得体会 |
| 分类 | 工作/学习/生活/项目/其他 |
| 优先级 | 高优先/中等/一般 |
| 标签 | 自定义标签，最多5个 |
| 颜色 | 6种苹果系统色彩可选 |
| 进度 | 0-100%，默认100% |
| 日期 | 开始日期和完成日期 |

### 分类颜色

| 分类 | 颜色代码 |
|------|---------|
| 工作 | 🔵 蓝色 #0A84FF |
| 学习 | 🟣 紫色 #BF5AF2 |
| 生活 | 🟢 绿色 #30D158 |
| 项目 | 🟠 橙色 #FF9F0A |
| 其他 | ⚪ 灰色 #8E8E93 |

## 🎨 设计系统

### 色彩规范
采用苹果系统色彩，克制使用：
- **蓝色** - 主色调，强调交互
- **绿色** - 成功、完成状态
- **橙色** - 提醒、收获
- **紫色** - 学习、创意
- **红色** - 错误、删除

### 排版规范
- **字体** - Inter / SF Pro / 系统默认
- **标题** - 20-24px，600字重
- **正文** - 15-16px，400字重
- **辅助** - 12-14px，400字重

### 间距系统
- **基础单位** - 4px
- **小间距** - 8px
- **中间距** - 16px
- **大间距** - 24-32px

## 🔧 自定义

### 修改配色
编辑 `css/style.css` 文件中的 CSS 变量：

```css
:root {
    --blue: #0071E3;
    --green: #34C759;
    --orange: #FF9500;
    /* ... */
}
```

### 添加分类
在 `js/app.js` 中找到 `categoryLabels` 对象添加新分类。
