# 功能调试指南

## 问题1: 心得体会入口

心得体会字段已经存在于表单中，位置在"工作内容"下方。

## 问题2: 里程碑添加到下周计划

### 测试步骤

1. **打开测试页面**
   ```
   http://localhost:8080/test_milestone.html
   ```
   查看里程碑周数计算是否正确

2. **打开浏览器控制台**（Cmd+Option+J）

3. **添加工作项**
   - 点击"添加工作"
   - 填写信息
   - 添加里程碑，日期设为**下周**
   - 保存

4. **查看控制台日志**
   应该看到：
   ```
   当前周: 2026 年 26 周
   下周: 2026 年 27 周
   里程碑数量: 1
   里程碑 0 : 测试里程碑 日期: 2026-06-30 周: 2026 年 27 周
   ✅ 属于下周，准备添加
   ✅ 已添加到下周计划
   保存下周数据，添加了 1 条
   ```

5. **跳转到下周查看**
   - 点击右上角下一周按钮
   - 查看"下周计划"列表
